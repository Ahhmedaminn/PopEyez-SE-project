## Shared Status Values
- User Status: Active, Deactivated
- Event Status: Planned, Ongoing, Completed, Cancelled
- Task Status: Pending, In Progress, Done
- Booking Request Status: Pending, Approved, Declined, Counter Proposal
- Sourcing Request Status: Pending, Accepted, Declined
- Delivery Status: Preparing, Out for Delivery, Delivered, Delayed
- Invoice Status: Pending Review, Approved, Paid, Rejected
- RSVP Status: Attending, Not Attending, Maybe, No Response
- Message Status: Sent, Received, Seen
- Check-In Status: Not Arrived, Arrived
